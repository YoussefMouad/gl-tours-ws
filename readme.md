## Utilisation
Fixer votre addresse IP à : 192.168.1.244
Démarer le serveur MongoDB.
Lancer les commandes suivantes pour la première foix: 
  - `composer install`
  - `php artisan migrate`

Et puis lancer le serveur : 
  - `php artisan serve --host=192.168.1.244 --port=8000`
